What is Turtle Lab?

ما هو Turtle Lab؟

Turtle Lab is a simple coding playground where children write Python Turtle commands and instantly see their drawings come to life. It is inspired by the classic Logo learning experience, but uses modern Python-style code.

Turtle Lab هو ملعب برمجي بسيط يكتب فيه الأطفال أوامر Python Turtle ثم يشاهدون رسوماتهم تظهر مباشرة. الفكرة مستوحاة من تجربة Logo التعليمية القديمة، ولكن باستخدام أسلوب بايثون الحديث.

Educational Background

الخلفية التعليمية

Turtle Lab teaches programming through drawing. Instead of starting with abstract examples, children give commands to a turtle, then see the result immediately. This makes concepts like sequence, repetition, angles, variables, and functions easier to understand.

يعلّم Turtle Lab البرمجة من خلال الرسم. بدلًا من البدء بأمثلة مجردة، يعطي الطفل أوامر للسلحفاة ثم يرى النتيجة فورًا. هذا يجعل مفاهيم مثل ترتيب الأوامر، والتكرار، والزوايا، والمتغيرات، والدوال أسهل في الفهم.

⭐

Inspired by Logo

مستوحى من Logo

Turtle graphics became famous through Logo, an educational programming language designed to help learners understand programming and geometry visually. Python Turtle follows the same idea using Python syntax.

اشتهرت رسومات السلحفاة من خلال لغة Logo التعليمية، التي ساعدت المتعلمين على فهم البرمجة والهندسة بصريًا. يستخدم Python Turtle الفكرة نفسها ولكن بأسلوب بايثون.

🐢

Visual learning first

التعلم البصري أولًا

Children see every command affect the turtle, so code feels like a creative tool instead of a list of rules.

يرى الأطفال تأثير كل أمر على السلحفاة، لذلك تبدو البرمجة أداة إبداعية بدلًا من كونها مجموعة قواعد فقط.

What Language Will Kids Learn?

ما اللغة التي سيتعلمها الأطفال؟

Turtle Lab teaches the basics of Python, one of the most widely used modern programming languages. Children write real Python-style instructions such as loops, variables, and functions, while Turtle graphics makes the results visual and fun.

يعلّم Turtle Lab أساسيات لغة Python، وهي من أشهر لغات البرمجة الحديثة. يكتب الأطفال أوامر بأسلوب بايثون الحقيقي مثل الحلقات، والمتغيرات، والدوال، بينما تجعل رسومات السلحفاة النتيجة مرئية وممتعة.

t.forward(100)
t.right(90)
t.color("blue")

for i in range(4):
    t.forward(100)
    t.right(90)

Technology Used

التقنيات المستخدمة

The current app version uses standard web technologies, Python Turtle-style syntax, Skulpt for running Python in the browser, and browser drawing surfaces such as HTML Canvas or SVG.

يستخدم الإصدار الحالي من التطبيق تقنيات الويب الأساسية، وأسلوب Python Turtle، ومكتبة Skulpt لتشغيل Python داخل المتصفح، ومساحات رسم في المتصفح مثل HTML Canvas أو SVG.

Python

Python is the programming language introduced through the lessons. Learners use simple Python-style code to control the turtle.

Python هي لغة البرمجة التي يتعرّف عليها المتعلمون من خلال الدروس. يستخدم الأطفال أوامر بسيطة بأسلوب بايثون للتحكم في السلحفاة.

Requirements to Run Turtle Lab

متطلبات تشغيل Turtle Lab

Turtle Lab runs in a modern web browser. In most cases, children do not need to install Python or any programming tools.

يعمل Turtle Lab داخل متصفح ويب حديث. في معظم الحالات، لا يحتاج الأطفال إلى تثبيت Python أو أي أدوات برمجية.

✅

Basic Requirements

المتطلبات الأساسية

  • A laptop, desktop, tablet, or Chromebook
  • A modern web browser
  • JavaScript enabled
  • Enough screen space to view the editor and drawing area
  • Internet connection if the app loads libraries from a CDN
  • No Python installation required for the browser version
  • جهاز كمبيوتر أو جهاز لوحي أو Chromebook
  • متصفح ويب حديث
  • تفعيل JavaScript
  • مساحة شاشة مناسبة لعرض محرر الكود ومنطقة الرسم
  • اتصال بالإنترنت إذا كان التطبيق يحمّل المكتبات من CDN
  • لا حاجة لتثبيت Python في نسخة المتصفح
🌐

Does it need internet?

هل يحتاج إلى الإنترنت؟

If Turtle Lab uses online library links, internet is required when opening the app. If all library files are downloaded and included locally, the app can work offline after setup.

إذا كان Turtle Lab يستخدم روابط مكتبات من الإنترنت، فسيحتاج إلى اتصال بالإنترنت عند فتح التطبيق. أما إذا تم تنزيل كل ملفات المكتبات وإضافتها محليًا، فيمكن أن يعمل التطبيق دون إنترنت بعد الإعداد.

🐍

Do I need to install Python?

هل أحتاج إلى تثبيت Python؟

No. The browser version is designed to run Python-style Turtle code inside the web page. Learners only need to open the website.

لا. نسخة المتصفح مصممة لتشغيل كود Python Turtle داخل صفحة الويب. يحتاج المتعلم فقط إلى فتح الموقع.

Recommended Browsers

المتصفحات الموصى بها

For the best experience, use the latest version of one of these browsers:

لأفضل تجربة، استخدم أحدث إصدار من أحد هذه المتصفحات:

Google Chrome Microsoft Edge Mozilla Firefox Safari ChromeOS / Chromebook browser

Older browsers may not support all features, especially the code editor, drawing canvas, or browser-based Python runtime.

قد لا تدعم المتصفحات القديمة جميع الميزات، خصوصًا محرر الكود أو منطقة الرسم أو تشغيل Python داخل المتصفح.

For Parents and Teachers

للأهل والمعلمين

Turtle Lab is designed to make programming visual, playful, and less intimidating. It introduces real programming ideas such as sequencing, loops, variables, and functions through drawing and experimentation.

تم تصميم Turtle Lab ليجعل البرمجة مرئية وممتعة وأقل صعوبة. يعرّف الأطفال على مفاهيم برمجية حقيقية مثل ترتيب الأوامر، والحلقات، والمتغيرات، والدوال من خلال الرسم والتجربة.

This app is best used for guided exploration. Children should be encouraged to change numbers, colors, and shapes to discover how code affects the result.

من الأفضل استخدام هذا التطبيق من خلال الاستكشاف الموجّه. شجّع الأطفال على تغيير الأرقام والألوان والأشكال لاكتشاف تأثير الكود على النتيجة.
